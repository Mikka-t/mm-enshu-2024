<h2>いちごタルト</h2>

<h3>材料</h3>
<ul>
  <li>グラニュー糖: 150g</li>
  <li>牛乳: 200ml</li>
  <li>バニラエッセンス: 少々</li>
  <li>小麦粉: 200g</li>
  <li>いちご: 適量</li>
  <li>砂糖: 100g</li>
  <li>バター: 50g</li>
  <li>卵黄: 2個</li>
  <li>生クリーム: 200ml</li>
</ul>

<h3>作り方</h3>
<ol>
  <li>
    <h4>ホイップクリームを作る</h4>
    <ul>
      <li>生クリームとグラニュー糖をホイップする。</li>
      <li>砂糖を加えてさらにホイップする。</li>
    </ul>
  </li>
  <li>
    <h4>タルト生地を作る</h4>
    <ul>
      <li>小麦粉、バター、砂糖、卵黄を混ぜる。</li>
      <li>混ぜた生地を焼いて、焼いたタルト生地を作る。</li>
    </ul>
  </li>
  <li>
    <h4>カスタードクリームを作る</h4>
    <ul>
      <li>牛乳と砂糖を煮る。</li>
      <li>卵黄とバニラエッセンスを加える。</li>
    </ul>
  </li>
  <li>
    <h4>タルトを作る</h4>
    <ul>
      <li>
        <h4>タルト生地に詰める</h4>
        <ul>
          <li>焼いたタルト生地にカスタードクリームを詰める。</li>
        </ul>
      </li>
      <li>
        <h4>飾る</h4>
        <ul>
          <li>カスタードクリームを詰めたタルトにホイップクリームといちごを飾る。</li>
        </ul>
      </li>
    </ul>
  </li>
</ol>
```
